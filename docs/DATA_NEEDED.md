# SafeETA 실데이터 연동에 필요한 자료

## 꼭 필요한 자료

1. 전국 초정밀 버스 `노선 기본정보` 샘플 응답
2. 전국 초정밀 버스 `노선 경유지` 샘플 응답
3. 전국 초정밀 버스 `실시간 위치` 샘플 응답
4. 시연 대상 도시 1곳
5. 시연 대상 노선 2~3개
6. 목표 정류장 이름 또는 ID

## 응답에서 확인하고 싶은 필드

- 노선 ID
- 차량 ID 또는 차량번호
- 수집 시각
- 위도 / 경도
- 속도
- 방향각
- 정류장 ID
- 정류장 이름
- 정류장 순번
- GNSS / GPS 수신 타입
- 운행 이벤트 또는 상태값

## 신호등 데이터까지 붙일 때 추가로 필요한 것

1. 교차로 기본정보 샘플 응답
2. 신호제어기 잔여시간 샘플 응답
3. 시연 정류장 주변 교차로 매칭 기준

## 데모 전략

- 처음에는 도시 1곳만 붙여도 충분합니다.
- 공모전에서는 전국형 구조를 설명하고, 실제 시연은 대표 지역으로 좁히는 편이 안전합니다.
- 핵심은 데이터 종수를 넓히는 것보다 `탑승 가능성 판단`이 실제로 동작하는 것입니다.
